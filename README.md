# 🌍 İz Analiz | Karbon Ayak İzi Hesaplayıcı

Bu benim web geliştirme dünyasına adım attığım **ilk projem!** 🚀 

## 📋 Proje Hakkında
İz Analiz, kullanıcıların günlük yaşam alışkanlıklarını (ulaşım, beslenme, ısınma vb.) analiz ederek yıllık yaklaşık karbon salınımlarını hesaplayan interaktif bir web uygulamasıdır. 

### ✨ Özellikler
- **Modern Arayüz:** Glassmorphism ve karanlık mod temalı şık tasarım.
- **Etkileşimli Test:** 10 soruluk hızlı ve kolay analiz süreci.
- **Kişiselleştirme:** Kullanıcı ismiyle hitap eden dinamik yapı.

## 🛠️ Kullanılan Teknolojiler
- **HTML5** (Yapısal kurgu)
- **CSS3** (Inter fontu, Flexbox ve modern görsel efektler)
- **JavaScript** (Hesaplama mantığı ve LocalStorage yönetimi)
